Panel Installation

Set up Pterodactyl Panel from scratch on a fresh Ubuntu/Debian server. Run commands in order as root.

💡 Requires a fresh Ubuntu 20.04+ or Debian 11+ server. Make sure you have root access before starting.
Step 1 · Switch to Root
bash 
sudo su
Step 2 · Update Package Lists
bash 
apt update
Step 3 · Upgrade Packages
bash 
apt upgrade
Step 4 · Install curl
bash 
apt install curl
Step 5 · Full System Update
bash 
sudo apt update -y && sudo apt upgrade -y
Step 6 · Configure Firewall Rules
bash 
sudo ufw allow mysql &&
sudo ufw allow OpenSSH &&
sudo ufw allow ssh &&
sudo ufw allow 80 &&
sudo ufw allow 443 &&
sudo ufw allow 2022 &&
sudo ufw allow 8080
Step 7 · Enable Firewall
bash 
sudo ufw enable
Step 8 · Run Pterodactyl Installer
bash 
bash <(curl -s https://pterodactyl-installer.se/)
